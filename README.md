# Go Wiki

Print articles from Wikipedia from the command line

## Install

`go get github.com/thedouglenz/go-wiki/...`

## Usage

```
wiki <search term>
# <search term> is any string (multiple words can be space separated)
```

## Example

```
$ wiki batman

Batman

Batman is a fictional superhero ...
```
